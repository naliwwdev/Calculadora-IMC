# Calculadora-IMC
Este projeto consiste no desenvolvimento de uma Calculadora de IMC (Índice de Massa Corporal) com o objetivo de aplicar conceitos básicos de lógica de programação, entrada e validação de dados, cálculos matemáticos e apresentação de resultados ao usuário.

A aplicação permite que o usuário informe peso (em quilogramas) e altura (em metros). A partir desses dados, o sistema realiza automaticamente o cálculo do IMC utilizando a fórmula padrão reconhecida pela Organização Mundial da Saúde (OMS):

IMC = peso / (altura²)

Após o cálculo, o programa exibe o valor do IMC e sua classificação correspondente, como:
baixo peso, peso normal, sobrepeso ou obesidade (graus I, II e III).

O projeto tem caráter educacional, sendo utilizado para testar, praticar e consolidar conhecimentos fundamentais de programação, como:

Operadores matemáticos
Estruturas condicionais
Interação com o usuário
Organização e clareza do código

Além disso, o sistema pode ser expandido futuramente com melhorias, como interface gráfica, histórico de resultados, validações mais avançadas ou integração com banco de dados.
